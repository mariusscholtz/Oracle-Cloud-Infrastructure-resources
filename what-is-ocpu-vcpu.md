## What is Oracle OCPU, and VCPU?

Overview: 
Oracle OCPU or Oracle Compute Unit or Oracle Compute Processing Unit = provides the CPU capacity equivalent to one physical core of an Intel Xeon processor with hyperthreading enabled, In this example one core = one OCPU.  <sup>[(1)</sup>
